ASIS Case Study

Angular User Management System

Developed by Arshia Shahidi for ASIS recruitment process

Project Overview

This is a technical case study developed for ASIS recruitment process to evaluate Angular fundamentals including component structure, API integration, and routing capabilities.

Core Features

  • User listing with cards
  • User detail pages
  • Search functionality
  • Interactive maps

Tech Stack

  • Angular 19
  • TypeScript
  • Tailwind CSS
  • JSONPlaceholder API
